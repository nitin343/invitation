# Invitation v2 — Cinematic Scroll Section

This repo contains **one** cinematic scroll experience section built with **React + Framer Motion** (Vite).

## Setup

1. Install deps:
   - `npm i`
2. Start:
   - `npm run dev`

## Images (required)

Place these files in `public/`:

- `public/hands.jpg` (memory layer)
- `public/couple.jpg` (hero layer)

If either image is missing, a subtle gradient fallback renders so you can still preview the motion.
